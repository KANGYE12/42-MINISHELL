
# minishell

*This project was created as part of the 42 curriculum by <kanye> <isrguerr>.*

## Description

minishell is a simplified implementation of a Unix shell. The goal is to create a functional command interpreter that allows executing programs, managing processes, and understanding the fundamental concepts of how modern shells work.

## Instructions

### Compilation
```bash
make
```

### Execution
```bash
./minishell
```

### Cleanup
```bash
make clean
make fclean
```

## Features

- External command execution
- Redirections (`>`, `<`, `>>`, `<<`)
- Pipes (`|`)
- Environment variables
- Signals (SIGINT, SIGQUIT)
- Command history
